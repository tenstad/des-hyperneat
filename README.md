# DES-HyperNeat

The DES-HyperNeat framework and three implementations (layered, single cppn, and coevolutional), implemented in go.  
This work was originally part of the [Deep Evolvable-Substrate HyperNEAT](https://hdl.handle.net/11250/2739318) master’s thesis in computer science at [NTNU](https://www.ntnu.edu/idi).
